# web_backend
这里是网站后端 

使用git及IDEA版本控制功能管理



### 运行本项目：

1. 使用任何方式下载本项目
2. 在IDEA中打开
3. 修改\src\main\resources\application.properties 文件中的数据库 url、用户名及密码
4. 使用Maven编译 pom.xml文件
5. 使用MySQL运行javadb.sql文件
6. 启动项目
